# commerce-project

Java 콘솔 환경에서 동작하는 간단한 **커머스 플랫폼**입니다.  
카테고리/상품/장바구니/주문/관리자 모드를 지원하며, **람다 & 스트림을 활용한 상품 조회/필터링 기능**을 포함하고 있습니다.

---

## 주요 기능 정리

### 1. 메인 메뉴

- 카테고리 목록 출력
- 장바구니/주문 관련 메뉴 (장바구니가 비어있지 않을 경우)
- 관리자 모드 진입
- 프로그램 종료

### 2. 카테고리별 상품 조회 & 가격 필터링 (람다 & 스트림 사용)

- 전체 상품 보기
- 100만원 이하 상품 보기
- 100만원 초과 상품 보기

### 3. 상품 선택 → 장바구니 추가 흐름

- 조회한 상품을 선택
- 선택한 상품을 장바구니에 추가

### 4. 장바구니 & 주문 처리
- 장바구니에 담긴 상품과 수량을 출력
- 총 주문 금액 계산
- 주문 확정 / 주문 삭제 / 메인으로 돌아가기 선택 가능

### 5. 등급(Grade)에 따른 할인 적용
- Grade enum (예: BRONZE, SILVER, GOLD, PLATINUM)
- Grade.fromNumber(1~4)로 등급 선택
- grade.getDiscountRate()를 활용하여 할인 금액 계산

### 6. 관리자 모드
#### 6-1. 상품 추가
- 카테고리를 선택한 후
- 상품명, 가격, 설명, 재고 수량 입력
- 확인 후 해당 카테고리에 상품 추가

### 6-2. 상품 수정
- 상품명을 입력하여 대상 상품 조회
- 수정 항목(가격/설명/재고수량)을 선택하여 수정

#### 6-3. 상품 삭제
- 상품명을 입력하여 해당 상품 삭제

#### 6-4. 전체 상품 현황
- 모든 카테고리의 상품을 돌면서 출력

### 프로젝트 구조
````
commerce-console-project/
├─ README.md
├─ .gitignore
└─ src
    └─ main
        └─ java
            └─ com
                └─ enolj
                    └─ commerce
                        ├─ CommerceSystem.java   # 전체 흐름 제어 (메인 메뉴, 장바구니, 관리자 모드 등)
                        ├─ Main.java             # 프로그램 시작점 (CommerceSystem 생성 및 실행)
                        ├─ Category.java         # 카테고리 도메인 (이름, 상품 리스트)
                        ├─ Product.java          # 상품 도메인 (이름, 가격, 설명, 재고, 출력 메서드 등)
                        ├─ Customer.java         # 고객 도메인 (필요 시 확장)
                        └─ Grade.java            # 고객 등급 Enum (BRONZE, SILVER, GOLD, PLATINUM)
````
